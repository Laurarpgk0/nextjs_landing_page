## Getting Started

# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/Laurarpgk0/nextjs_landing_page.git
```

```bash
# Install Dependencies
$ yarn

# Run Aplication
$ yarn start
```
Go to http://localhost:3000/ to see the result.
